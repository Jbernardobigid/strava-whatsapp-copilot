# TrainingBuddy Documentation

This folder contains project documentation for the Strava WhatsApp Copilot / TrainingBuddy app.

Recommended reading order:

1. `PROJECT_STATE.md` — current source of truth
2. `RUNBOOK.md` — how to operate and troubleshoot
3. `ARCHITECTURE.md` — how the app is structured
4. `DECISIONS.md` — why key choices were made
5. `ROADMAP.md` — future work
6. `SESSION_LOG.md` — chronological history

These files are intended to be committed to GitHub and updated at the end of major work sessions.
