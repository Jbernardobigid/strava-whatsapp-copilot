# Strava WhatsApp Copilot

A beginner-friendly prototype that connects to Strava, reads cycling activities, and sends WhatsApp messages in Brazilian Portuguese with simple training insights.

## Current features

- Strava OAuth connection
- Persistent Strava token storage
- Fetch recent Strava activities
- Send WhatsApp messages through Twilio Sandbox
- PT-BR ride summaries
- Basic ride interpretation and next-day suggestions

## Stack

- Python
- FastAPI
- Twilio WhatsApp Sandbox
- Strava API

## Notes

This is currently a local prototype for learning and iteration.
